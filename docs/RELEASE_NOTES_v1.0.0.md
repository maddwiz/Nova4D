# Nova4D v1.0.0

Release date: 2026-02-20

## Highlights

- Initial Nova4D release for Cinema 4D automation.
- Queue bridge server with API key auth and rate limiting.
- 60+ endpoint surface across scene/material/mograph/xpresso/animation/render/io/system.
- Cinema 4D Python plugin worker (`plugins/Nova4D/nova4d_plugin.pyp`).
- Headless render launcher endpoint (`POST /nova4d/batch/render`).
- OpenClaw extension (`extensions/openclaw/cinema4d-bridge`).
- Python SDK (`python-sdk/nova4d.py`) and MCP server (`mcp-server/nova4d_mcp.py`).
- Packaging pipeline output `Nova4D-v1.0.0.zip`.
